# 桥接设计模式，将抽象和实施分离开来，通过桥梁进行链接。
说白了就是接口 + 实现类  
客户端使用的时候，除了初始化的时候，其他的全部避免和具体的实现类交互。  
也就是说你的接口需要包含全部的功能和逻辑。
实现类可以有不同套，A B C 
类似于数据库sdk的设计。

我们使用桥接
只要定义好抽象层，实现层随意扩展。

写代码的时候先考虑抽象层定义，不考虑具体实现层。因为实现层你想怎么实现就怎么实现。
